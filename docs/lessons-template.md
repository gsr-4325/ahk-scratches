# Lessons snapshot

This file is intended to be overwritten by report-publishing workflows.

## What it should contain

- a short status line for the latest run
- links or paths to the latest report files
- a stable list of known pitfalls that future agents should not rediscover

## Source of truth for persistent notes

The durable notes live in:

- `AGENTS.md`
- `docs/agent-pitfalls.md`

The generated lessons snapshot should summarize, not replace, those files.
